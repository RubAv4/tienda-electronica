# Arquitectura del proyecto

- Backend: Spring Boot (carpeta `Backend`).
- Frontend: Angular (carpeta `FrontAngular/electronica-franko`).
- Base de datos: MySQL (configuración en `application.properties`).

## Módulos principales

- Autenticación y roles (admin, reponedor, usuario).
- Gestión de productos, categorías y stock.
- Carrito de compras, pedidos y pedidos por usuario.
